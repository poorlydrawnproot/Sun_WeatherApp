# Weather For.... The Sun??
## Yes!*
### *Not really
This app generates random numbers within the range of the temperature of the sun's surface (The photosphere), which is about 5800 kelvin!
I originally was learning F#, but switched courses to C#, and with that, I installed Visual Studio and spent a few hours fiddling around and looking at syntax reference guides, I finished my first ever C# app, Sun_WeatherApp. 
I made this to help myself learn C# for Hackclub's Summer event of 2026, Stardance!
